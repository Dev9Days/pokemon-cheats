# 기술 제거 치트

공통으로 `시스템/기타 마스터 코드`가 필요합니다.

```text
D8BAE4D9 4864DCE5
```

아래 코드는 모두 `L+SELECT`를 동시에 누르면 실행됩니다.

## 1. 직접 삭제형

선택한 파티 포켓몬의 선택한 기술을 즉시 삭제합니다. 내부적으로 기존 `MoveDeleterForgetMove` 함수를 호출하므로 기술을 지운 뒤 뒤쪽 기술을 앞으로 당기고 PP 보너스도 정리합니다.

주의: 버튼을 길게 누르면 같은 슬롯 기준으로 반복 실행되어 여러 기술이 연속 삭제될 수 있습니다. `L+SELECT`는 짧게 한 번만 누르고, 실행 후 치트를 끄는 쪽이 안전합니다. 이 방식은 알, 기술 1개만 보유, 마지막 파도타기 포켓몬 제한을 우회합니다.

6줄 구조입니다.

```text
3CDE8216 3875A404
[포켓몬 슬롯 줄]
[기술 슬롯 줄]
3CDE8216 3875A404
[빌드별 함수 포인터 하위 줄]
[빌드별 함수 포인터 상위 줄]
```
